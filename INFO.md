# MoodLights

Easy mood-based light management for Home Assistant.

## Features

- Mood entities with presets
- Dynamic light groups
- State save and restore
- Exclusion rules
- Optional confirmation workflow

## Installation

Install via HACS or manually copy to `custom_components/moodlights`.
