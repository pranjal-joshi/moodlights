# MoodLights

Easy mood-based light management for Home Assistant.

## Features

- Mood entities with Activate and Revert buttons
- Per-light settings: brightness, colour temperature, RGB colour
- State save and restore (up to 3 snapshots per mood)
- Callable HA services for automations

## Installation

Install via HACS or manually copy to `custom_components/moodlights`.
